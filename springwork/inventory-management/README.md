# Inventory Management Spring Boot Project

This project is a Spring Boot application for managing inventory across multiple companies. It includes basic setup for entities, REST APIs, and H2 in-memory database for development.

## How to Run

1. Make sure you have Maven and Java 17+ installed.
2. Run `mvn spring-boot:run` in the project directory.
3. Access H2 console at `/h2-console`.

## Features
- Multi-company inventory management
- CRUD APIs for Company, Product, Inventory, and User
- Spring Data JPA & H2 database
